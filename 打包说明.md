# 四川新数录像批量下载器 - 打包说明

## 快速开始

### 方法1：一键打包（推荐）

运行以下命令，自动完成所有打包步骤：

```bash
package.bat
```

这会：
1. 编译Python程序为exe
2. 复制所有必要的dll文件
3. 复制Java组件
4. 创建完整的发布目录

输出目录：`dist\四川新数录像批量下载器\`

### 方法2：生成安装程序

如果已经运行过`package.bat`，可以生成专业的安装程序：

```bash
build-installer.bat
```

这会：
1. 检查发布目录
2. 使用Inno Setup生成安装程序

输出文件：`installer\四川新数录像批量下载器-安装程序.exe`

### 方法3：手动打包

1. **编译exe**
   ```bash
   cd hikvision_downloader
   pyinstaller --onefile --windowed --name "四川新数录像批量下载器" main.py
   ```

2. **收集文件**
   手动收集以下文件到同一目录：
   - `dist\四川新数录像批量下载器.exe`
   - Java组件（`hikvision_java\bin\*.jar`）
   - 海康SDK dll（见下文）

3. **打包分发**
   压缩整个目录为zip文件

## 分发方式

### 方式1：绿色版（推荐）
- 使用`package.bat`生成的目录
- 直接压缩为zip文件
- 用户解压后直接运行exe

优点：
- 无需安装
- 可便携使用
- 文件透明

### 方式2：安装程序
- 使用`build-installer.bat`生成
- 专业的安装向导
- 自动创建快捷方式

优点：
- 用户体验好
- 自动关联文件
- 易于卸载

## 常见问题

### 1. 缺少DLL文件

现象：程序启动失败，提示缺少dll

解决：
- 运行`package.bat`会自动复制dll
- 或手动从SDK目录复制

### 2. Inno Setup未找到

现象：`build-installer.bat`报错

解决：
- 下载安装Inno Setup: https://jrsoftware.org/isinfo.php
- 安装时勾选"Add to PATH"

### 3. 编译失败

现象：PyInstaller编译错误

解决：
- 确保Python 3.8+
- 安装所有依赖: `pip install -r requirements.txt`
- 安装PyInstaller: `pip install pyinstaller`

### 4. 安装后无法运行

现象：安装后程序闪退

解决：
- 检查是否安装了Visual C++ Redistributable
- 检查杀毒软件是否误删dll文件
- 以管理员身份运行
