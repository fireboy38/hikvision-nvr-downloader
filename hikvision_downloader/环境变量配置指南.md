# Java和Python环境变量配置指南

## Windows系统环境变量配置

### 一、查找安装路径

首先确认Java和Python的安装路径：

#### 1. 查找Python安装路径
在命令行中运行：
```bash
where python
```
或者在PowerShell中：
```powershell
Get-Command python | Select-Object -ExpandProperty Source
```

常见安装位置：
- `C:\Users\{用户名}\AppData\Local\Programs\Python\Python3x\`
- `C:\Python3x\`
- `C:\Program Files\Python3x\`

#### 2. 查找Java安装路径
在命令行中运行：
```bash
where java
```
或者在PowerShell中：
```powershell
Get-Command java | Select-Object -ExpandProperty Source
```

常见安装位置：
- `C:\Program Files\Java\jdk-xx\bin\`
- `C:\Program Files\Java\jre-xx\bin\`
- `C:\Program Files\Eclipse Adoptium\jdk-xx-hotspot\bin\`

### 二、添加到系统环境变量

#### 方法1：通过系统设置（推荐）

1. **打开环境变量设置**
   - 右键点击"此电脑"或"我的电脑"
   - 选择"属性"
   - 点击"高级系统设置"
   - 点击"环境变量"按钮

2. **编辑Path变量**
   - 在"用户变量"或"系统变量"区域找到"Path"变量
   - 选中"Path"，点击"编辑"
   - 点击"新建"，添加以下路径：
     - Python安装路径（如：`C:\Python310\` 或 `C:\Users\{用户名}\AppData\Local\Programs\Python\Python310\`）
     - Python Scripts路径（如：`C:\Python310\Scripts\`）
     - Java的bin目录（如：`C:\Program Files\Java\jdk-17\bin\`）

3. **验证配置**
   - 打开新的命令行窗口
   - 运行：
     ```bash
     python --version
     java -version
     ```

#### 方法2：使用PowerShell命令（自动添加）

```powershell
# 添加Python到Path
$userPath = [Environment]::GetEnvironmentVariable("Path", "User")
$pythonPath = "C:\Python310"
$pythonScriptsPath = "C:\Python310\Scripts"

if ($userPath -notlike "*$pythonPath*") {
    [Environment]::SetEnvironmentVariable("Path", "$userPath;$pythonPath;$pythonScriptsPath", "User")
    Write-Host "Python已添加到用户Path"
} else {
    Write-Host "Python已在Path中"
}

# 添加Java到Path
$userPath = [Environment]::GetEnvironmentVariable("Path", "User")
$javaPath = "C:\Program Files\Java\jdk-17\bin"

if ($userPath -notlike "*$javaPath*") {
    [Environment]::SetEnvironmentVariable("Path", "$userPath;$javaPath", "User")
    Write-Host "Java已添加到用户Path"
} else {
    Write-Host "Java已在Path中"
}

# 刷新环境变量
$env:Path = [System.Environment]::GetEnvironmentVariable("Path","User") + ";" + [System.Environment]::GetEnvironmentVariable("Path","Machine")
```

**注意**：请将上述脚本中的路径替换为您实际的安装路径。

### 三、为项目配置环境变量（针对本海康NVR下载器）

如果不想修改系统环境变量，可以在项目中配置：

#### 1. 在代码中指定路径

**Python SDK路径** (`core/hiksdk_downloader.py`):
```python
# 设置SDK路径
SDK_PATH = r"C:\Users\Administrator\CH-HCNetSDKV6.1.6.45_build20210302_win64_20210508181836\CH-HCNetSDKV6.1.6.45_build20210302_win64\库文件"
os.add_dll_directory(SDK_PATH)
os.chdir(SDK_PATH)
```

**Java SDK路径** (`core/java_downloader.py`):
```python
import os
import subprocess

# 设置Java路径（如果不在Path中）
JAVA_HOME = r"C:\Program Files\Java\jdk-17"
java_exe = os.path.join(JAVA_HOME, "bin", "java.exe")

# 调用Java程序
subprocess.run([java_exe, "-jar", "your-app.jar"])
```

#### 2. 使用虚拟环境（推荐）

### 五、常见问题

#### 1. 命令找不到
- **原因**：环境变量未生效或路径错误
- **解决**：
  - 关闭所有命令行窗口，重新打开
  - 检查路径是否正确
  - 使用完整路径测试（如：`C:\Python310\python.exe`）

#### 2. 多个版本冲突
- **原因**：Path中有多个Python/Java版本
- **解决**：
  - 将需要的版本路径移到Path最前面
  - 使用Python Launcher（py命令）：`py -3.10`
  - 在代码中明确指定版本

#### 3. 权限问题
- **原因**：修改系统变量需要管理员权限
- **解决**：
  - 以管理员身份运行命令行
  - 或者修改用户变量而非系统变量

### 六、快速配置脚本

保存为 `setup_env.bat` 并以管理员身份运行：

```batch
@echo off
echo 正在配置Python和Java环境变量...

:: 设置路径（请根据实际情况修改）
set PYTHON_PATH=C:\Python310
set PYTHON_SCRIPTS=C:\Python310\Scripts
set JAVA_PATH=C:\Program Files\Java\jdk-17\bin

:: 添加到系统Path
setx PATH "%PATH%;%PYTHON_PATH%;%PYTHON_SCRIPTS%;%JAVA_PATH%" /M

echo 环境变量配置完成！
echo 请重新打开命令行窗口以使更改生效。
pause
```

## 总结

1. **优先配置用户环境变量**（无需管理员权限）
2. **验证路径正确性**（使用where命令）
3. **重启命令行**（使环境变量生效）
4. **测试命令**（python --version, java -version）

如遇问题，请检查：
- 路径是否存在
- 路径中是否包含特殊字符
- 是否有多个版本冲突
- 是否需要管理员权限

**提示**：为了项目的稳定性，建议使用虚拟环境或项目中明确指定路径。
